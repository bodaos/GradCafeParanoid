Gradcafeparanoid

An app for people obsessively checking their grad school application result through GradCafe

Download this project as a .zip file Download this project as a tar.gz file

During the graduate school application season, applicants are obsessively refreshing the GradCafe result search page to check if anyone hear from the programs that they apply it. This app makes that experience possible on the mobile and much more "addictive".

If you need support for this application, you can email me lis@carleton.edu
